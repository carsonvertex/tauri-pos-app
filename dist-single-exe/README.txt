# Tauri POS Application - Standalone Package

## Quick Start
1. Double-click start-pos.bat to launch the application
2. Or run start-pos.ps1 in PowerShell for better error handling

## What's Included
- **TauriPOS.exe** - Main application executable
- **backend.jar** - Spring Boot backend server
- **data/** - SQLite database directory
- **start-pos.bat** - Windows batch launcher
- **start-pos.ps1** - PowerShell launcher (recommended)

## Requirements
- Windows 10/11
- Java 17+ (will be bundled in full package)

## Features
- ??Offline-first POS system
- ??SQLite local database
- ??Automatic backend startup
- ??Cross-platform Tauri frontend
- ??Data synchronization ready

## Troubleshooting
- If Java errors occur, ensure Java 17+ is installed
- Check that ports 8080 and 1420 are available
- Run as administrator if permission issues occur

## Support
For issues or questions, check the application logs in the console window.
